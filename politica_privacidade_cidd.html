Política de Privacidade

Última atualização: 25/09/2024 11:03AM

1. Informações Coletadas

1.1. Informações de Rede

O aplicativo solicita permissão para acessar o estado da rede Wi-Fi e verificar se há conexão com a Internet. Esta funcionalidade é necessária para enviar imagens para nossa API hospedada em um servidor externo e receber as previsões geradas pelo modelo de Inteligencia Artificial.

1.2. Informações da Câmera

O aplicativo utiliza a câmera do dispositivo para capturar imagens do cacau que serão enviadas para a análise do modelo de Inteligencia Artificial. Não armazenamos as imagens capturadas nos nossos servidores.

O aplicativo pode utilizar o flash da câmera para melhorar a qualidade das imagens capturadas, especialmente em condições de baixa luminosidade.

1.3. Armazenamento de Dados

O aplicativo solicita permissão para acessar o armazenamento interno do dispositivo para permitir que o usuário selecione imagens previamente salvas para envio e análise.

As previsões geradas pelo modelo de IA e o histórico de capturas realizadas pelo usuário são armazenados localmente no dispositivo usando o AsyncStorage. Estes dados não são compartilhados com terceiros e permanecem no dispositivo do usuário.

1.4. Informações Compartilhadas

O aplicativo permite que o usuário compartilhe um link para download do aplicativo com outras pessoas. Nenhuma informação pessoal é transmitida durante esse processo.

1.5. Processamento de Imagens

As imagens capturadas ou selecionadas são enviadas para nossa API Flask hospedada em https://api-cidd.npca.tec.br/predict para serem analisadas pelo modelo de IA treinado. Nenhuma outra informação pessoal é enviada junto com as imagens. A API responde com a previsão relacionada à presença de doenças no cacau.

2. Como Usamos as Informações

As informações coletadas são utilizadas para:

  • Processamento de Imagens: Analisar imagens do cacau e fornecer previsões sobre a presença de possíveis doenças.
  • Histórico de Capturas: Armazenar localmente as previsões e as imagens associadas para consulta futura pelo próprio usuário.
  • Conectividade: Verificar a conectividade com a Internet para enviar imagens ao servidor e obter previsões.

3. Compartilhamento de Informações

Não compartilhamos informações pessoais com terceiros, exceto nos seguintes casos:

  • Com Provedores de Serviços: Enviamos imagens para nossa API externa apenas para fins de processamento. Nenhuma informação pessoal adicional é compartilhada com a API.

4. Segurança das Informações

Adotamos medidas técnicas e organizacionais adequadas para proteger as informações coletadas contra acesso, uso, alteração e divulgação não autorizados. No entanto, nenhum método de transmissão pela Internet ou de armazenamento eletrônico é 100% seguro e, portanto, não podemos garantir segurança absoluta.

5. Direitos dos Usuários

Você tem os seguintes direitos em relação aos seus dados:

  • Acessar: Você pode solicitar o acesso ao histórico de capturas armazenado no aplicativo.
  • Excluir: Você pode excluir o histórico de capturas e as previsões armazenadas no aplicativo a qualquer momento.
  • Revogar Permissões: Você pode revogar as permissões concedidas ao aplicativo, como acesso à câmera e ao armazenamento interno, nas configurações do seu dispositivo.

6. Alterações nesta Política de Privacidade

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que você revise esta política regularmente para estar ciente de qualquer alteração. Notificaremos você sobre quaisquer alterações publicando a nova política nesta página.
